# מדד התשואות — Brand Rules

## Project Identity
- **Name:** מדד התשואות
- **Author:** אור זמר — מתכנן פיננסי
- **Purpose:** Premium analytics tool for comparing Israeli financial instruments

## Visual Identity
- **Background:** `#0A0A0A` (Matte Black)
- **Surface:** `#111111` (cards), `#161616` (elevated)
- **Accent Primary:** `#D4AF37` (Gold)
- **Accent Light:** `#F0D060` (Gold hover/shimmer)
- **Accent Muted:** `#8B7320` (Gold dark)
- **Text Primary:** `#F5F5F5`
- **Text Secondary:** `#9CA3AF`
- **Text Muted:** `#4B5563`
- **Positive:** `#22C55E` (Green — gains)
- **Negative:** `#EF4444` (Red — losses)
- **Border:** `rgba(212,175,55,0.15)`

## Typography
- **Font Family:** Heebo, Assistant, sans-serif
- **Direction:** RTL (Hebrew)
- **Heading XL:** 2rem / 800 weight
- **Heading L:** 1.5rem / 700
- **Body:** 0.875rem / 400
- **Caption:** 0.75rem / 300

## Component Rules
- **Cards:** `border-radius: 16px`, `border: 1px solid rgba(212,175,55,0.15)`
- **Hover state:** gold border glow `box-shadow: 0 0 24px rgba(212,175,55,0.25)`
- **Transitions:** `0.2s ease` for all interactive elements
- **Tables:** alternating rows `rgba(255,255,255,0.02)`, header gold-tinted

## Data Sources
- GemelNet (גמל נט) — primary reference
- Data is illustrative and updated manually
- Always show disclaimer on data pages
